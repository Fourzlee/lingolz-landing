# lingolz-landing

Minimum viable landing page for lingolz.com, sufficient for Meta Business verification.

## Stack

- Single static `index.html` with inline CSS
- No JavaScript, no external dependencies
- Hosted on Cloudflare Pages, deployed automatically on push to `main`
- Custom domain: `lingolz.com`

## Local preview

Open `index.html` in any browser. There's no build step.

## Deploy

Pushes to `main` auto-deploy via Cloudflare Pages. No CI configuration needed.
